# 🏨 Hospitality Data Analysis Dashboard (Excel Project)

##  Project Overview
This project is an end-to-end Hospitality Data Analysis Dashboard built using Microsoft Excel.  
The objective of this project is to analyze hotel performance metrics such as revenue, bookings, occupancy, cancellations, and customer ratings.

The dashboard provides interactive insights using Pivot Tables, Pivot Charts, Slicers, and KPI Cards.

---

##  Business Objective
To analyze hotel performance across:
- Cities
- Hotel Types
- Room Classes
- Booking Status
- Weekday vs Weekend trends

And identify:
- Revenue trends
- Occupancy patterns
- Cancellation impact
- Customer satisfaction levels



##  Key Performance Indicators (KPIs)

- 💰 Total Revenue  
- 🏨 Total Bookings  
- 📈 Occupancy %  
- ❌ Cancellation %  
- ⭐ Average Rating  

All KPIs are dynamically connected using Pivot Tables and GETPIVOTDATA formulas.

---

##  Dashboard Features

- Revenue Trend (Monthly)
- Revenue by City (Descending order)
- Revenue by Hotel
- Booking Status Analysis (Donut Chart)
- Room Class Performance
- Weekday vs Weekend Analysis
- Interactive Slicers (City, Year, Hotel Type)

---

##  Tools & Techniques Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- GETPIVOTDATA
- Custom Number Formatting (K/M format)
- Dashboard Design using Shapes
